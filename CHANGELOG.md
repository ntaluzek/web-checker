## [0.1.0] - 2025-11-25

### Added
- URL list management with multiple named lists
- Bookmark folder integration for dynamic URL lists
- List reordering support (drag and drop or move up/down)
- Manual list editing and deletion
- Auto-progression through URLs with keyboard shortcuts
- Progress tracking per list (resume from where you left off)
- Visual progress indicator showing current position (e.g., "Resume 3/10")
- Next URL navigation keyboard shortcut (default: Alt + Arrow Right)
- Popup interface with Start/Resume buttons per list
- Options page for managing URL lists
- Empty list indicators for lists with no URLs
- Tooltip showing next URL on hover
- Floating navigation box on visited pages with Next/Restart button
- Customizable keyboard shortcuts
- Toggle for shortcut reminder tooltips
- Manifest V3 implementation
